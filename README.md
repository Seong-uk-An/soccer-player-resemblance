밤에 코딩 다 끝난 후 - github에 올리는 기본 명령어:

1. git init
   1-1. git remote add origin { 깃허브 주소 } ( 최초 한 번만 하면 됨. )

2. git add .

3. git commit -m "{ 버전 }"

4. git push origin { 너의 브랜치 (브랜치 이름은 sebin 으로 지을 거 ㅋ) }
